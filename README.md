# 🕹️ 레트로 점프 (Retro Jump)

원버튼으로 즐기는 레트로 아케이드 점프 러너 웹게임.

**▶ 플레이하기: https://hindyaaa.github.io/RetroJump/**

## 조작법

| 입력 | 동작 |
|---|---|
| 스페이스 / ↑ / 화면 탭 | 점프 (길게 누르면 더 높이) |
| M | 배경 음악 켜기/끄기 |

절벽을 뛰어넘고 코인을 모으세요. 떨어지면 게임 오버!

## 특징

- 320×180 저해상도 도트 그래픽 + CRT 스캔라인 효과
- WebAudio로 실시간 생성되는 칩튠 배경 음악과 8비트 효과음
- 점점 빨라지는 절차적 지형 생성, 최고 기록 저장
- PWA 지원 — 폰 브라우저에서 "홈 화면에 추가"하면 앱처럼 전체 화면·오프라인으로 실행

외부 라이브러리·에셋 없이 `index.html` 파일 하나로 동작합니다.
